# DISTANCE-BETWEEN-TWO-POINTS

## AIM:
To write a python program to find the distance two 2 points
## ALGORITHM:
### Step 1: 
import numpy
### Step 2: 
input l1
### Step 3:
input l2 

### Step 4: 
Substitute the values in the distance formula 
![formula](formula.jpg)
### Step 5: 
print the output
### PROGRAM:
~~~
import numpy as np
l1=[4,2]
l2=[10,6]
dist=np.sqrt(((l2[0]-l1[0])**2)+((l2[1]-l1[1])**2))
print("{:.2f}.format(dist))
~~~
